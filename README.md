Besides pdflatex, compile-time dependencies include:

* algorithm2e.sty
* hyperref.sty
* complexity.sty

On Ubuntu, the necessary packages are:

* texlive-latex-base
* texlive-science

To compile this article, run 

    ./compile-paper.sh

The output is equivalence.pdf.
